注0：该游戏要求 WIN10 日文环境，可用 Locale Emulator 代替系统转区。
（汉化直接运行 AGE_CHS.EXE 即可，不需要转区）

注1：安装版比较繁琐（内有安装教程），但保证可用；绿色版有微小几率无法运行。
（希望有用绿色版的热心人士反馈一下~）

注2：若打开游戏后出现“未响应”，请在启动前设置中选择“软件模拟”，而非硬件加速。（要求高性能CPU）
或者尝试 Win8 兼容模式运行。

注3：绿色版运行后显示版本仍为 1.0，但实际上已安装 1.03 升级档。若不放心请点此以展开手动升级教程。
（实际上是注册表问题，用管理员权限运行“第一次游戏前运行.bat”即可解决）
  1. 前往蓝奏云（提取码: eru）下载“天结2注册表修复.bat”、“amayui_lm_103.exe”两个文件。（另外两个是AP01档即特典，以及旧版升级档。）
  2. 将“天结2注册表修复.bat”移动到游戏根目录下（即与“AGE.EXE”同一文件夹），双击运行。（由于升级档根据注册表中的路径安装，无法手动选择，所以要先在注册表中加入正确路径。）
  3. 右键“amayui_lm_103.exe”，以 Locale Emulator -> Run in Japanese (Admin) 方式运行，中途只需回车就行。
  4. 最后如果弹出 PlayDRM 验证窗口，请打开“破解备份”中的“AmaLaby Crak.zip”，将其中的文件覆盖到游戏路径即可。
