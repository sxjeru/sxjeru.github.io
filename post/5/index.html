天結いラビリンスマイスター （天结2）


官网：https://www.eukleia.co.jp/eushully/eu021.html 

本体1.01+AP01——

安装版（资源来源点此）：https://bit.ly/3yDqeg0 

绿色版（解压即玩）：https://bit.ly/3BDwciK 

（运行“AGE.EXE”即可开玩。）

注0：该游戏要求 WIN10 日文环境，可用 Locale Emulator 代替系统转区。

注1：安装版比较繁琐（内有安装教程），但保证可用；绿色版有微小几率无法运行。（希望有用绿色版的热心人士反馈一下~）

注2：绿色版运行后显示版本仍为 1.0，但实际上已安装 1.01 升级档。若不放心请点此以展开手动升级教程。
  1. 前往蓝奏云（提取码: eru）下载“天结2注册表修复.bat”、“amayui_lm_101.exe”两个文件。（另外一个是AP01档，也就是特典）
  2. 将“天结2注册表修复.bat”移动到游戏根目录下（即与“AGE.EXE”同一文件夹），双击运行。（由于升级档根据注册表中的路径安装，无法手动选择，所以要先在注册表中加入正确路径。）
  3. 右键“amayui_lm_101.exe”，以 Locale Emulator -> Run in Japanese (Admin) 方式运行，中途只需回车就行。

至此，便更新至 1.01 了。

CG 与 BGM 依旧待续——
